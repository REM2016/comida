#form>: Define el formulario.
action="mailto:tu_correo@ejemplo.com": Especifica que el formulario enviará los datos al correo electrónico indicado. Reemplaza "tu_correo@ejemplo.com" con tu dirección de correo.
method="post": Indica que los datos serán enviados usando el método POST.
enctype="text/plain": Especifica el tipo de contenido de los datos enviados.
#label> y #input>: Crean los campos del formulario:
type="text": Campo de texto simple para el nombre.
type="email": Campo de texto específico para direcciones de correo electrónico.
required: Hace que el campo sea obligatorio.
#textarea>: Crea un área de texto para el mensaje.
#button>: Crea un botón para enviar el formulario.
